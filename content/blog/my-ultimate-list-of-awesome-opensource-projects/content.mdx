This is a constantly work-in-progress list of cool project I have come across or closely worked with.<br />It solves for me the *"I remember there was a cool project, but the name completely slipped my mind! Halp!"* case.

## Sooo,

### Make 2D, 3D, VR / AR visualisation

* [Mywebar](https://mywebar.com/) <Comment>In-browser AR</Comment>
* [Potree](https://github.com/potree/potree/) <Comment>WebGL point cloud viewer for large datasets</Comment>
* [Sketchfab](https://sketchfab.com/) <Comment>A marketplace for 3D models</Comment>
* [ThreeJS](https://threejs.org/) <Comment>A library for making in-browser 3D graphic</Comment>
* [D3JS](https://d3js.org/) <Comment>A library for data visualization</Comment>

### Do machine learning

* [Openpilot](https://github.com/commaai/openpilot) <Comment>Open source driver assistance system</Comment>
* [Google Tesseract](https://opensource.google/projects/tesseract) <Comment>an OCR library</Comment>

### Emulate

* [Puppeteer](https://github.com/puppeteer/puppeteer) <Comment awesome>headless Chrome emulator</Comment>
* [Playwright](https://github.com/microsoft/playwright) <Comment>Cross-browser API automation</Comment>

### Debug your app  

* [chrome-remote-interface](https://github.com/cyrus-and/chrome-remote-interface) <Comment>Chrome Debugging Protocol interface for different environments</Comment>

### Parse something

* [Cheerio](https://www.npmjs.com/package/cheerio) <Comment>HTML parser</Comment>

### Compile something

* [Pkg](https://github.com/vercel/pkg) <Comment>Compile your Node app into binary</Comment>

### Highlight syntax

* [PrismJS](https://prismjs.com/)

### Create an editor

* [DraftJS](https://draftjs.org/) <Comment>Medium-like editor framework</Comment>
* [Monaco Editor](https://microsoft.github.io/monaco-editor/) <Comment>In-browser source code editor</Comment>

## Develop for mobile

* [Flutter](https://flutter.dev/)

## Process some images

* [ImageMagic](https://imagemagick.org/index.php) <Comment>Good article <a href="https://hackernoon.com/save-time-by-transforming-images-in-the-command-line-c63c83e53b17">here</a></Comment>
* [Sharp](https://sharp.pixelplumbing.com/)

## Work with a database

* [TypeORM](https://github.com/typeorm/typeorm)
* [Prisma](https://www.prisma.io/)
* [Minimongo](https://www.npmjs.com/package/minimongo) <Comment>Client-side Mongo backed by IndexedDb</Comment>
