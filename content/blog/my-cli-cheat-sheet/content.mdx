## Git

### History

```bash
# display branching tree
git log --graph --full-history --all --pretty=format:"%h%x09%d%x20%s";
git log --graph --decorate --oneline;
# display history of commits for a specific branch
git log --walk-reflogs master;
# un-stage all uncommitted changes
git reset HEAD
# un-stage all uncommitted changes and erase them
git reset HEAD --hard
# remove un-tracked files
git clean -fd
git clean -fX
```

### Work with tags

```bash
# get all remote tags
git fetch --all --tags
# kill remote tag
git push --delete origin v1.2.0
# kill local tag
git tag -d v1.2.0
```

## Bash

```bash
# parametrized alias
alias dssh='function __sampleAlias(){ docker exec -it $1 /bin/bash; };__sampleAlias'
# launch chrome from the command line in Mac
alias chrome="/Applications/Google\\ \\Chrome.app/Contents/MacOS/Google\\ \\Chrome"
# get console command by its PID
ps -p [PID] -o args
# find where a binary is installed
which [CLI_COMMAND]
# list all files sorting by date, human readable size
ls -ltah
# ... or reverse
ls -ltahr
# ... or by size
ls -lSah
# ... or by size reverse
ls -lSahr
# copy a file from a remote host 123.123.123.123 over ssh working on port 55332
scp -P 55332 admin@123.123.123.123:/home/remote/path/to/file.gz ./
# zip a folder
zip -r folder.zip folder/
```

## Yarn

```bash
# make a total package upgrade
yarn upgrade <package-name> --major
# find out why the package was installed
yarn why <package-name>
# get the latest version of a package that exists on NPM
yarn info <package-name> | grep version:
```

## Docker

```bash
# build an image
docker build -t vendor-name/image-name .
# run an image
docker run vendor-name/image-name
# run an image (e.g. a mongodb image) in daemon mode
docker run --name some-mongo -d mongo:latest
# "ssh" into a container with ID 9a54dd26acb2
docker exec -it 9a54dd26acb2 bash
# see all running containers
docker ps
# see logs of a container with id 9a54dd26acb2
docker logs 9a54dd26acb2
# copy file "config.txt" into a container with ID 9a54dd26acb2 into /root folder
docker cp ./config.txt 9a54dd26acb2:/root/
# remove all exited containers
docker rm $(docker ps -a -f status=exited -q)
# remove all working containers
docker rm $(docker ps -a -q)
# remove all images
docker rmi $(docker images -a -q)
# remove volumes, networks
docker system prune
```

## Mongo

~~~bash
# dump a database (will create dump/[collections])
mongodump --db database_name
# import a database (will import all collections in dump/ folder)
mongorestore dump
~~~
